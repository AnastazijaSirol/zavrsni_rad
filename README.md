# Završni rad
Izrada predikcijskog modela iz skladišta podataka za detekciju prijevara

## Ključne faze: 
1. Eksplorativna analiza podataka
2. Izrada relacijskog modela podataka (MySQL) i dimenzijskog modela podataka (Apache Spark)
3. Provedba ETL procesa za integraciju i obradu podataka iz više izvora
4. Vizualizacija podataka pomoću Tableau dashboard-a
5. Izrada predikcijskog modela za detekciju prijevara temeljenog na XGBoost algoritm
